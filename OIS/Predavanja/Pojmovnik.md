# Pojmovnik — Osnove informacijskih sistemov

Združeni pomembni pojmi iz vseh predavanj (P1–P7), abecedno urejeni, brez podvajanj.

| Pojem                                | Definicija                                                                                                                                          |
| ------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1. NO**                            | Prva normalna oblika — atomarni atributi, brez ponavljajočih skupin                                                                                 |
| **2. NO**                            | Druga normalna oblika — 1. NO + brez parcialnih (delnih) odvisnosti                                                                                 |
| **3. NO**                            | Tretja normalna oblika — 2. NO + brez tranzitivnih odvisnosti                                                                                       |
| **AS-IS / TO-BE**                    | Analiza obstoječega stanja procesa (AS-IS) in načrtovanje želenega stanja (TO-BE)                                                                   |
| **ASP**                              | Application Service Provider — ločena instanca aplikacije za vsakega naročnika                                                                      |
| **BA**                               | Business Analytics — analiza zgodovinskih podatkov za razvoj ukrepov in priporočil                                                                  |
| **BI**                               | Business Intelligence — analitični sistemi za pridobivanje vpogledov iz podatkov                                                                    |
| **Big Data (3V)**                    | Volume (obseg), Velocity (hitrost), Variety (raznolikost)                                                                                           |
| **BPI**                              | Business Process Improvement — postopne, inkrementalne izboljšave procesov                                                                          |
| **BPM**                              | Business Process Management — sistematično upravljanje procesov kot neprestani cikel                                                                |
| **BPR**                              | Business Process Reengineering — radikalna prenova poslovnega procesa                                                                               |
| **Business Continuity**              | Zagotavljanje neprekinjenega poslovanja ob resnih motnjah                                                                                           |
| **Balanced Scorecard**               | Okvir, ki povezuje strategijo s KPI-ji prek 4 perspektiv                                                                                            |
| **BYOD**                             | Bring Your Own Device — uporaba osebnih naprav zaposlenih za delo                                                                                   |
| **CIM**                              | Computer Integrated Manufacturing — računalniško integrirana proizvodnja                                                                            |
| **CLTV**                             | Customer Lifetime Value — življenjska vrednost stranke za organizacijo                                                                              |
| **COTS**                             | Commercial Off-The-Shelf — kupljena gotova programska rešitev                                                                                       |
| **CRM**                              | Customer Relationship Management — strategije, procesi in IS za upravljanje odnosov s strankami                                                     |
| **CSF**                              | Critical Success Factor — ključni faktor za uspeh organizacije                                                                                      |
| **DaaS**                             | Desktop as a Service — najem virtualnega namizja v oblaku                                                                                           |
| **Dark data**                        | Podatki, ki jih organizacija zbira, a ne analizira ali izkorišča                                                                                    |
| **Data Cleansing**                   | Čiščenje podatkov — odkrivanje in odpravljanje napak in podvojitev                                                                                  |
| **Data Governance**                  | Sistematično upravljanje podatkov — pravila, procesi, odgovornosti za kakovost, dostop in deljenje                                                  |
| **Data Mart**                        | Manjša, oddelčna različica podatkovnega skladišča                                                                                                   |
| **Data Quality**                     | Kakovost podatkov — točnost, konsistentnost, ažurnost                                                                                               |
| **Data Steward**                     | Skrbnik podatkov — odgovoren za kakovost in skladnost podatkov                                                                                      |
| **Data Warehouse (DW)**              | Centralni repozitorij zgodovinskih podatkov za analitiko in odločanje                                                                               |
| **Dashboard**                        | Vizualni prikaz KPI-jev in CSF-jev za hiter pregled stanja                                                                                          |
| **DBMS**                             | Database Management System — programska oprema za upravljanje podatkovne baze                                                                       |
| **DevOps**                           | Pristop, ki združuje razvoj in operacije za hitrejše in zanesljivejše uvajanje sprememb                                                             |
| **Diferenciacija**                   | Strategija ponujanja edinstvenih izdelkov ali storitev, ki jih konkurenca ne more enostavno posnemati                                               |
| **Digitalizacija**                   | Avtomatizacija obstoječih procesov (isto, a hitreje)                                                                                                |
| **Digitalna preobrazba**             | Sprememba poslovnega modela z uporabo tehnologije (drugačen način dela)                                                                             |
| **Disaster Recovery**                | Postopki za obnovo IT infrastrukture po nesreči                                                                                                     |
| **Disruptivna inovacija**            | Inovacija, ki radikalno spremeni trg in izrine obstoječe igralce                                                                                    |
| **DMAIC**                            | Define, Measure, Analyze, Improve, Control — metodologija za izboljšavo procesov                                                                    |
| **Enterprise Architecture**          | Celostna poslovno-informacijska arhitektura, ki povezuje poslovno poslanstvo s podatkovno, aplikacijsko in tehnološko arhitekturo                   |
| **ER diagram**                       | Entity-Relationship diagram — grafični prikaz entitet, atributov in relacij                                                                         |
| **ERP**                              | Enterprise Resource Planning — integriran sistem, ki povezuje oddelke prek skupne podatkovne baze                                                   |
| **ERP II**                           | Razširjen ERP, ki poleg jedrnih modulov vključuje SCM, CRM, BI in e-poslovanje                                                                      |
| **ETL**                              | Extract, Transform, Load — postopek prenosa in preoblikovanja podatkov v skladišče                                                                  |
| **FAIS**                             | Functional Area Information System — IS za posamezno funkcionalno področje (finance, HR, marketing ...)                                             |
| **FaaS / Serverless**                | Function as a Service — izvajanje posameznih funkcij brez upravljanja strežnikov                                                                    |
| **Funkcionalna odvisnost**           | Vrednost enega atributa enolično določa vrednost drugega                                                                                            |
| **Ganttov diagram**                  | Časovni prikaz nalog projekta z vidom na trajanje, zaporednost in vire                                                                              |
| **HTML**                             | Označevalni jezik za prikaz vsebine na spletnih straneh                                                                                             |
| **HRIS**                             | Human Resource Information System — IS za upravljanje človeških virov                                                                               |
| **Human-in-the-loop**                | Pristop, kjer človek ohrani nadzor in odgovornost nad odločitvami UI                                                                                |
| **IaaS**                             | Infrastructure as a Service — najem infrastrukture (strežniki, omrežja, shranjevanje)                                                               |
| **IS**                               | Socio-tehnični sistem za zbiranje, obdelavo, shranjevanje in distribucijo informacij                                                                |
| **IT**                               | Tehnološki del IS (strojna + programska oprema + omrežja)                                                                                           |
| **IT infrastruktura**                | Pet ključnih komponent (strojna oprema, programska oprema, upravljanje podatkov, telekomunikacije, tehnološke storitve), ki tvorijo platformo za IS |
| **IT strateški načrt**               | Dolgoročni načrt IT, ki podpira poslovno strategijo, usmerja arhitekturo in razporeja vire                                                          |
| **JOIN**                             | Operacija, ki poveže dve tabeli prek skupnega polja                                                                                                 |
| **Kaj-če analiza**                   | Preverjanje občutljivosti rezultata na spremembe vhodnih parametrov                                                                                 |
| **Kardinalnost**                     | Število primerkov ene entitete, povezanih z drugo (1:1, 1:N, M:N)                                                                                   |
| **KM**                               | Knowledge Management — sistematično upravljanje znanja v organizaciji                                                                               |
| **KPI**                              | Key Performance Indicator — ključni kazalnik uspešnosti                                                                                             |
| **Kognitivna pristranskost**         | Sistematična napaka v presoji, ki vpliva na odločanje                                                                                               |
| **Konkurenčna prednost**             | Sposobnost podjetja, da dosega nižje stroške, višjo kakovost ali večjo hitrost od konkurence                                                        |
| **Kvantno računalništvo**            | Računalništvo na podlagi qubitov, ki so lahko hkrati v stanju 0 in 1 (superpozicija)                                                                |
| **Low-code / No-code**               | Platforme za razvoj aplikacij brez ali z minimalnim programiranjem                                                                                  |
| **MDM (Master)**                     | Master Data Management — upravljanje glavnih/referenčnih podatkov organizacije                                                                      |
| **MDM (Mobile)**                     | Mobile Device Management — upravljanje mobilnih naprav v organizaciji                                                                               |
| **MIS**                              | Management Information System — poročanje za taktično raven                                                                                         |
| **Medfunkcijski proces**             | Proces, ki poteka čez več funkcionalnih področij in zahteva usklajevanje oddelkov                                                                   |
| **Metapodatki**                      | Podatki o podatkih — opisujejo vsebino, strukturo in kontekst                                                                                       |
| **Model Drift**                      | Postopno poslabšanje natančnosti modela UI zaradi spreminjanja razmer                                                                               |
| **Množična prilagoditev**            | Izdelava prilagojenih izdelkov v masovnem obsegu (ang. *mass customization*)                                                                        |
| **N-tier arhitektura**               | Večnivojska arhitektura z ločenimi plastmi (odjemalec, spletni strežnik, aplikacijski strežnik, podatkovna baza)                                    |
| **NPV**                              | Net Present Value — neto sedanja vrednost; primerja diskontirane prihodnje denarne tokove z začetno investicijo                                     |
| **NoSQL**                            | "Not only SQL" — sistemi za velike količine nestrukturiranih podatkov                                                                               |
| **Normalizacija**                    | Postopek razgradnje tabel za odpravo redundance in anomalij                                                                                         |
| **OLAP**                             | Online Analytical Processing — analitične poizvedbe nad večdimenzionalnimi (ali tabelaričnimi) podatki                                              |
| **OLTP**                             | Online Transaction Processing — obdelava tekočih transakcij                                                                                         |
| **Omrežni učinek**                   | Pojav, ko vrednost platforme narašča s številom uporabnikov                                                                                         |
| **Opisna analitika**                 | Kaj se je zgodilo? (retrospektiva)                                                                                                                  |
| **Outsourcing**                      | Zunanje izvajanje IT storitev pri zunanjem izvajalcu                                                                                                |
| **PaaS**                             | Platform as a Service — najem platforme (vključno z OS) za razvoj aplikacij                                                                         |
| **PERT diagram**                     | Mrežni diagram nalog z odvisnostmi; identificira kritično pot projekta                                                                              |
| **PLM**                              | Product Lifecycle Management — upravljanje življenjskega cikla izdelka                                                                              |
| **Podatkovna kocka**                 | Večdimenzionalna struktura za analizo podatkov po več dimenzijah hkrati                                                                             |
| **Podatkovni / informacijski silos** | Izoliran nabor podatkov, dostopen le enemu oddelku; povzroča nekonsistentnost in podvajanje                                                         |
| **Porterjev model 5 sil**            | Okvir za analizo rivalstva, grožnje vstopov, moči kupcev, moči dobaviteljev in grožnje nadomestkov                                                  |
| **Portfelj projektov**               | Celota vseh IT projektov v organizaciji; cilj je izbrati prave projekte                                                                             |
| **Poslovni proces**                  | Zbirka povezanih aktivnosti, ki pretvarjajo vhode v izhode in ustvarjajo vrednost                                                                   |
| **Predpisna analitika**              | Kaj naj naredimo? (priporočilo za ukrepanje)                                                                                                        |
| **Napovedna analitika**              | Kaj se bo zgodilo? (napoved)                                                                                                                        |
| **Primarni ključ (PK)**              | Atribut ali kombinacija atributov, ki enolično identificira vsak zapis v tabeli                                                                     |
| **Prototipiranje**                   | Iterativni pristop k razvoju IS z zgodnjim delujočim prototipom                                                                                     |
| **Pull model**                       | Dobavna veriga, ki temelji na dejanskih naročilih kupcev                                                                                            |
| **Push model**                       | Dobavna veriga, ki temelji na napovedih povpraševanja                                                                                               |
| **QBE**                              | Query by Example — vizualni vmesnik za poizvedbe brez pisanja SQL                                                                                   |
| **ROI**                              | Return on Investment — donosnost investicije                                                                                                        |
| **RPA**                              | Robotic Process Automation — avtomatizacija rutinskih nalog s programskimi roboti                                                                   |
| **Realna opcija**                    | Pristop, kjer fazna uvedba IS zmanjšuje tveganje — pilotni projekt omogoča odločanje na podlagi dejanskih rezultatov                                |
| **Redundanca**                       | Ponavljanje istih podatkov na več mestih; vodi v nekonsistentnost                                                                                   |
| **Robno računalništvo**              | Edge Computing — obdelava podatkov blizu vira za manj zakasnitve                                                                                    |
| **Run/Grow/Transform**               | Razporejanje IT proračuna: vzdrževanje, rast, preobrazba                                                                                            |
| **SaaS**                             | Software as a Service — večnajemniška arhitektura; najem končne aplikacije brez skrbi za infrastrukturo                                             |
| **SCE**                              | Supply Chain Execution — izvajanje procesov v dobavni verigi                                                                                        |
| **SCM**                              | Supply Chain Management — upravljanje dobavne verige od surovin do končnega kupca                                                                   |
| **SCP**                              | Supply Chain Planning — planiranje dobavne verige                                                                                                   |
| **SDLC**                             | Systems Development Life Cycle — klasičen zaporedni razvojni cikel IS                                                                               |
| **SECaaS**                           | Security as a Service — varnostne storitve v oblaku                                                                                                 |
| **SIS**                              | Strateški informacijski sistemi — IS, ki neposredno vplivajo na konkurenčni položaj podjetja                                                        |
| **SOA**                              | Service-Oriented Architecture — arhitektura z neodvisnimi storitvami, ki komunicirajo prek standardnih protokolov                                   |
| **SQL**                              | Structured Query Language — standardni jezik za poizvedovanje in upravljanje relacijskih baz                                                        |
| **S&OP**                             | Sales & Operations Planning — proces usklajevanja napovedi prodaje z operativnim načrtom                                                            |
| **Scalability**                      | Razširljivost — zmožnost sistema, da prenese povečano obremenitev                                                                                   |
| **Scoring model**                    | Model ocenjevanja alternativ na podlagi uteženih kriterijev                                                                                         |
| **Shadow Systems**                   | Neuradni sistemi (tabele, baze), ki jih zaposleni vzdržujejo mimo uradnega IT                                                                       |
| **Simonov model odločanja**          | Intelligence → Design → Choice → Implementation                                                                                                     |
| **Six Sigma**                        | Metodologija za zmanjševanje napak in variabilnosti v procesih                                                                                      |
| **Strukturirani podatki**            | Podatki z jasno definirano obliko, shranjeni v tabelah (vrstice in stolpci)                                                                         |
| **Delno strukturirani podatki**      | Podatki z nekaj strukture, a ne v klasičnih tabelah (npr. XML, spletne strani)                                                                      |
| **Nestrukturirani podatki**          | Podatki brez vnaprej določene oblike (besedilo, slike, video)                                                                                       |
| **Stroškovno vodstvo**               | Strategija doseganja najnižjih operativnih stroškov v panogi                                                                                        |
| **Survivorship Bias**                | Pristranskost preživelih — analiza le uspešnih primerov vodi v napačne zaključke                                                                    |
| **TCO**                              | Total Cost of Ownership — skupni stroški lastništva IT skozi celotno življenjsko dobo                                                               |
| **TMS**                              | Transportation Management System — sistem za upravljanje transporta                                                                                 |
| **TPS**                              | Transaction Processing System — sistem za zbiranje in obdelavo vsakodnevnih poslovnih transakcij                                                    |
| **TQM**                              | Total Quality Management — celovito upravljanje kakovosti                                                                                           |
| **Tacitno znanje**                   | Znanje, ki temelji na izkušnjah in intuiciji (težko ga je formalizirati)                                                                            |
| **Tehnični dolg**                    | Posledica zanemarjanja posodabljanja IT; kopiči se in sčasoma povzroča velike operativne težave                                                     |
| **Trojna omejitev**                  | Čas, stroški in obseg — tri medsebojno odvisne dimenzije vsakega projekta                                                                           |
| **Tuji ključ (FK)**                  | Atribut v tabeli, ki se sklicuje na primarni ključ druge tabele                                                                                     |
| **UCP**                              | Use Case Points — metoda za oceno obsega in stroškov razvoja IS na podlagi primerov uporabe                                                         |
| **Usklajenost IT in poslovanja**     | Zagotavljanje, da IT strategija podpira poslovno strategijo, cilje in procese                                                                       |
| **Ustvarjalno uničenje**             | Proces, v katerem nove tehnologije uničijo obstoječe poslovne modele                                                                                |
| **Vanilla ERP**                      | Standardna implementacija ERP brez prilagoditev                                                                                                     |
| **Vendor lock-in**                   | Odvisnost organizacije od enega ponudnika IS, ki otežuje menjavo                                                                                    |
| **Veriga blokov**                    | Porazdeljena knjiga transakcij v nespremenljivo povezanih blokih                                                                                    |
| **Virtualizacija**                   | Tehnologija za poganjanje več virtualnih sistemov na enem fizičnem strežniku                                                                        |
| **Vmesna tabela**                    | Tabela, ki razgradi M:N relacijo na dve 1:N relaciji                                                                                                |
| **Vrednostna veriga**                | Porterjev model primarnih in podpornih aktivnosti, ki ustvarjajo vrednost za stranko                                                                |
| **Razširjena veriga vrednosti**      | Vrednostna veriga, ki vključuje dobavitelje, partnerje in stranke                                                                                   |
| **WMS**                              | Warehouse Management System — sistem za upravljanje skladišča                                                                                       |
| **XML**                              | Označevalni jezik za opis pomena podatkov in izmenjavo med sistemi                                                                                  |
| **Zeleno računalništvo**             | Green Computing — zmanjševanje okoljskega vpliva IT z energijsko učinkovitimi praksami                                                              |
| **Buy-off-the-shelf**                | Nakup obstoječe aplikacije brez prilagoditev                                                                                                        |
| **Best-of-breed**                    | Pristop, kjer se za vsak modul izbere najboljšega ponudnika, a integracija je zahtevnejša                                                           |
| **Odjemalec/strežnik**               | Arhitekturni model, v katerem odjemalec skrbi za vmesnik, strežnik pa za podatke in storitve                                                        |
| **Eksplicitno znanje**               | Znanje, ki je zapisano in dokumentirano (navodila, priročniki)                                                                                      |
| **DSS**                              | Decision Support System — podpora analitičnemu odločanju                                                                                            |
| **ESS**                              | Executive Support System — nadzorne plošče za strateško raven                                                                                       |
| **Analitični CRM**                   | Del CRM, ki analizira podatke o strankah (segmentacija, CLTV, odliv)                                                                                |
| **Operativni CRM**                   | Del CRM, ki podpira vsakodnevno prodajo, marketing in storitve                                                                                      |
| **Učinek biča**                      | Bullwhip Effect — eksponentno povečevanje nihanj povpraševanja vzdolž dobavne verige navzgor                                                        |
| **Iskanje cilja**                    | Obratna kaj-če analiza — iščemo vhod za želen izhod                                                                                                 |
| **Črni labod**                       | Nepredvidljiv dogodek z velikim vplivom, ki ga za nazaj razlagamo kot predvidljivega                                                                |
